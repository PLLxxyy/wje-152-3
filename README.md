# 数码回收估价工具

一款纯前端的二手数码设备回收估价工具，支持主流手机、平板、笔记本、耳机、相机的在线估价。

## 技术栈

- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite 5
- **数据存储**：浏览器 localStorage（无后端）
- **样式**：纯 CSS 内联（index.html `<style>` 标签）

## 启动方式

```bash
npm install
npm run dev
```

启动后访问 http://localhost:5173

## 功能说明

### 设备选择
支持手机、笔记本、平板、耳机、相机五大品类，每个品类下预置了主流品牌和近两到三年的热门型号，采用品类 -> 品牌 -> 型号三级联动选择。

### 成色评估
从四个维度评估设备状况：
- **屏幕状况**：完美 / 轻微划痕 / 明显划痕 / 碎裂
- **外壳状况**：完美 / 轻微磕碰 / 明显磕碰 / 严重变形
- **电池健康度**：90%以上 / 80%-90% / 70%-80% / 70%以下
- **功能状态**：全部正常 / 部分异常 / 无法开机

### 自动估价
系统根据型号基准价和各维度成色折扣自动计算预估回收价范围（上下浮动 5%）。

### 提交回收单
确认估价后填写姓名、联系方式、取件地址和期望上门时间，提交回收单。

### 我的回收
查看所有回收单，状态分为待取件、检测中、已打款。点击订单查看估价明细和检测结果。

### 热门价格排行
首页展示回收价最高的 10 款设备排行，方便用户比较。
